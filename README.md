# 🌌 NeonJourneys - A Cyberpunk Travel Journal

NeonJourneys is a full-stack MERN application that allows users to document their travel memories in a highly immersive, futuristic interface. Users can securely upload visual data (photos), log their location grids, list travel companions, and attach embedded audio logs (Spotify/YouTube) to create a deeply personal travel databank.

## 🚀 Features

* **Secure Authentication:** JWT-based user login and registration system.
* **Full CRUD Databanks:** Create, Read, Update, and Purge travel memory fragments.
* **High-Speed Visual Uploads:** Direct image integration using Cloudinary's secure API.
* **Immersive Audio Logs:** Intelligent media embedder that converts standard YouTube or Spotify links into functional UI audio players.
* **Expanded Detail View:** Clickable memory cards that launch a full-screen, interactive datapad with uncropped images and logs.
* **Responsive Cyberpunk UI:** Custom neon-styled interface built with Tailwind CSS, featuring hover-states, animations, and a glowing grid aesthetic.

## 🛠️ Technology Stack

**Frontend:**
* React.js
* Vite
* Tailwind CSS
* React Router DOM

**Backend:**
* Node.js
* Express.js
* MongoDB (Atlas)
* Mongoose
* JSON Web Tokens (JWT) & Bcrypt
* Multer & Cloudinary (Image Pipeline)
